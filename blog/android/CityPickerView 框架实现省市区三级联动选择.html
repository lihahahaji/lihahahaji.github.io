# CityPickerView

实现城市三级选择

  1. 打开build.gradle(Module:app)文件，新加如下依赖:
dependencies {
    implementation 'liji.library.dev:citypickerview:5.2.2'
}

点击 sync now

  1. 申明对象并且预加载数据
public void init()
{
    // 申明对象
    myPickerView=new CityPickerView();
    // 预加载数据
    myPickerView.init(myCityPickerView.this);
    // 设置默认显示的省市区
    CityConfig myCityConfig=new CityConfig.Builder()
            .province("浙江省") // 设置默认显示的省份
            .city("杭州市")     // 设置默认显示的市
            .district("余杭区") // 设置默认显示的区
            .build();
    // 设置config
    myPickerView.setConfig(myCityConfig);

    // 获取EditText对象
    Edt_myCityPikerEditText=findViewById(R.id.Edt_myCityPikerEditText);

    // 设置监听事件
    Edt_myCityPikerEditText.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View v) {
            myPickerView.setOnCityItemClickListener(new OnCityItemClickListener() {
                @Override
                public void onSelected(ProvinceBean province, CityBean city, DistrictBean district) {
                    Edt_myCityPikerEditText.setText(province+"-"+city+"-"+district);
                }

                @Override
                public void onCancel() {
                    super.onCancel();
                }
            });
            myPickerView.showCityPicker();
        }
    });
}
  1. 如果出现报错 app:checkDebugDuplicateClasses 则在 gradle.properties 中添加以下内容
android.useAndroidX=true
android.enableJetifier=true
  1. 在点击EditText时会跳出键盘，避免的方法如下 EditText没获得焦点前，单击只能获得焦点，并不触发onclick事件。EditText获得焦点时，默认会弹出软键盘；点击EditText，默认不会弹出软键盘。倘若不想弹出软键盘，可添加Java代码：
final EditText editCity=findViewById(R.id.edit_city);

//点击监听
editCity.setOnClickListener(new View.OnClickListener() {
    @RequiresApi(api = Build.VERSION_CODES.LOLLIPOP)
    @Override
    public void onClick(View v) {
        //点击编辑框隐藏软键盘
        editCity.setShowSoftInputOnFocus(false);
    }
}

换一种思路，倘若不需要EditText的焦点功能而只设置点击事件，可以不用添加Java源码，而通过xml里EditText属性控制，禁用获焦android:focusableInTouchMode=“false”。由此，触及EditText自然也不会弹出软键盘了。：

android:focusableInTouchMode="false"
