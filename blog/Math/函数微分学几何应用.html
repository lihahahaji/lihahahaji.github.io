# 函数微分学的几何应用

# 1. 函数的极值点

极值点处不一定是连续的，但是要求极值点两侧的函数有定义，即极值点也可以是间断点。

求某函数的极值点：

  1. 求一阶导数 找到 （1）导数等于0的点 （2）不可导点 （3）分段函数的分段点
  2. 求这些点左右侧的导数值 来判断属于极大值还是极小值

# 2. 函数的拐点

拐点是 连续 曲线上的凹弧和凸弧的分界点，拐点不一定可导，且如果拐点二阶可导的话，其导函数的值一定为0 注意 拐点应表示为 (x,f(x))

求某函数的极值点：

  1. 求二阶导数为 0 的点 或者二阶导数不存在的点
  2. 求这些左右侧导数的导数值或者求该点的三阶导数的导数值

# 3. 函数的渐近线

铅锤渐近线 就是与 x 轴垂直的渐近线

lim⁡x→x0f(x)=∞\lim_{x\to x_0} f(x)=\inftylimx→x0​​f(x)=∞

水平渐近线 就是与 x 轴水平的渐近线

lim⁡x→∞f(x)=y\lim_{x\to \infin} f(x)=ylimx→∞​f(x)=y

斜渐进线 即 lim⁡x→+∞f(x)x=a1,lim⁡x→+∞[f(x)−a1x]=b1\lim_{x\to+\infin}\frac{f(x)}{x}=a_1 , \lim_{x\to+\infin}[f(x)-a_1x]=b_1limx→+∞​xf(x)​=a1​,limx→+∞​[f(x)−a1​x]=b1​

lim⁡x→−∞f(x)x=a2,lim⁡x→−∞[f(x)−a1x]=b2\lim_{x\to-\infin}\frac{f(x)}{x}=a_2 , \lim_{x\to-\infin}[f(x)-a_1x]=b_2limx→−∞​xf(x)​=a2​,limx→−∞​[f(x)−a1​x]=b2​

求某函数的渐近线：

  1. 先找 铅锤渐近线 （函数的无定义点）
  2. 再找 水平渐近线
  3. 再找 斜渐近线

# 4. 求最值

找以下几种点求函数值

  1. 导数为零的点（驻点）
  2. 不可导点（导函数不存在的点）
  3. 函数定义区间的端点 （如果是开区间的话 则求函数区间两端的单侧极限）

# 5.作函数的图形

  1. 先观察函数的奇偶性（若存在奇偶性可以少一半的计算量）
  2. 分别求一阶导数和二阶导数
  3. 求出以下点
    1. f(x)f(x)f(x) 无定义
    2. f′(x)=0f'(x) = 0f′(x)=0
    3. f′(x)f'(x)f′(x) 无定义
    4. f′′(x)=0f''(x)=0f′′(x)=0
    5. f′′(x)f''(x)f′′(x) 无定义
  4. 根据以上的点将定义域划分为若干个子区间
  5. 确定函数图形在各个子区间上的单调性和凹凸性
  6. 确定函数的极值点和拐点
  7. 确定函数的渐近线（如果有的话）
  8. 做出函数图形
