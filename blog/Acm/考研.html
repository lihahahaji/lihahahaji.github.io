# 重庆大学 大数据与软件学院

# 电子信息专硕

# 经验贴：

# 1. 2022重庆大学877软件工程考研经验贴

前言

经过一年的艰苦奋斗，4月初终于收到了拟录取通知，不枉费一年的努力，也很感谢一路上前辈们的指导以及朋友们的鼓励。本文的目的主要是为了给报考重大软院的学弟学妹提供一点小小的建议。先介绍一下自己的情况，本人本科末流一本，科班出身，有一点科研经历（其实大创我认为是不算科研经历的，除非是国家级），无比赛成果。自我感觉能被录取很大一部分原因是因为初试分高，初试分数404，复试分数不是很理想，一战上岸。但是有一点要指明，重大的复试是十分公平的，完全没有歧视本科的情况，只是可能比较卷，但起码努力是能够看到结果的。

每个人的基础和情况都不同，请大家不要盲目完全跟随任一篇经验贴，还是要结合自身，取之精华为己所用，希望大家听过我的分享后能够少走一些弯路。

择校

择校这一步我是从3月份开始一直到报名前一周才最终确定下来的。一开始我的目标学校是哈工大深圳校区，但架不住845三本大黑书参考书，之后将目光转向了东北大学，在东大的考研群里混迹许久，当时感觉群里全是大佬，我一个混子还是不要跟大佬们卷了，铁成炮灰，而且东大的专业课是三门，因为我数学的基础不是很好，需要在数学上花费更多的时间，而至此将目标定为了不歧视本科出身、专业课科目少、简单、地区公共课不压分的重庆大学。下面分享一下查找学校考研信息的几个方法：

  1. 学校官网：学校官网永远是了解该学校信息的最权威、最官方的途径。直接百度XXX大学研招网，查找历年考试信息、录取分数情况等。
  2. 学长学姐：通常这种咨询都是收费的，且学长学姐的联系方式不是很好找，我当时是在知乎和咸鱼找到了几位辅导的学长学姐，付费了解了一些重大的考研信息。
  3. 机构：有些机构会做一些免费的公开直播，有些机构则只提供昂贵的指导，不推荐。
  4. 研招网：研招网的信息不全，但是一般都会提供对应学校的招生信息网的链接。

资料

资料这东西不求多只求精，找到最适合你的老师和资料才是最重要的，不要东一头西一头，最后时间都花费在找资料上而没有学到真的东西。下面我把我所用到以及战友们推荐的资料分享一下。

  • 数学：

1、高数部分：基础不好的同学推荐强烈推荐武忠祥老师！武老师的风格属于十分稳健，重基础，跟下来完全不需要担心基础不扎实的问题。武老师的基础部分和强化部分可以一路跟下来并搭配每天做武老师在公众号的每日一题（强推），强化部分推荐外加上张宇老师的1000题。基础较好的同学，追求高分，我推荐张宇老师的全家桶，张宇老师上课幽默风趣，发撒思维强，能够增强自己举一反三的能力，如果跟张宇老师，习题则需要做1000、660、880。计算能力不强的同学推荐做杨超老师的《三大计算》，这本书做完能够大幅提升计算能力。至于汤老师，我当时做了一半的1800，听了几节汤老师的课，不做评价。最后冲刺阶段，做08年到22的真题，外加李林老师的6+4预测卷，有能力的同学可以做张宇老师的8+4，今年数二好像押中了一道题，可惜只做了两张张宇老师的卷子。

2、线代部分：不用选择，跟李永乐大帝即可，全程跟下，线代满分不是任何问题。

3、概率论部分：由于本人考的是数学二，咨询了一下考数一的战友，比较推荐的是王式安老师。

  • 英语：

1、单词：贯穿考研始终，从你开始下定决心考研的那一天到上考场的前一天，每一天都要背单词。我使用的是墨墨背单词软件，如果有同学比较擅长结合图片记忆单词，也推荐百词斩。个人不推荐纸质单词书，因为不好记录每天背单词的情况，所以单词复习效果不如软件。

2、阅读：在单词量足够的基础上，搞定长难句语法，我当时跟的是刘晓燕老师的长难句课，刘老师上课的风格不好接受，但其讲的内容还是实打实的好，推荐跟。搞定语法后，就是练习阅读了，推荐读外刊。

3、翻译：田静老师的翻译书。

4、作文：王江涛老师的考研英语作文。

5、真题：推荐黄皮书，排版很舒服。

  • 政治：徐涛老师的强化课、腿姐的《背诵手册》、肖秀荣老师的1000题、肖四肖八。
  • 专业课：877考的内容是C+数据库，C语言看谭浩强老师的《C语言程序设计》即可，数据库的参考书为王成良老师的《数据库技术及应用》，习题和真题是当时在一些学长处购买及网上查找整理的，有需要的学弟学妹可以联系我。

初试

政治：

第一阶段：暑假结束到11月份，看徐涛老师的强化课，在对应的教材上做好笔记，这时不需要花费大量时间记忆这些内容，只需要过一遍，把知识点弄懂，徐涛老师的讲解是十分到位的，有个大概的知识点印象即可。在学习强化课的过程中应该把对应的1000题的内容做完，B站有不少UP主的公众号都会免费提供这两部分的对应表格，可以参考。

第二阶段：11月到考试前，二刷1000题，尽量三刷1000题。推荐买两本1000题，二刷用纸质的书，三刷则使用一些微信公众号的小程序，插空就刷两道，差不多一周就能刷完一遍。在11末开始，腿姐和涛哥会更新带背环节，推荐跟，搭配多刷1000题会让你牢牢记住考频高的知识点，选择题拿高分。

英语：

我考研英语二分数为86分，我的英语基础是六级擦边过那种。首先是单词，在开始决定考研的时候背单词的任务就开始了，从3月份到5月份，我用的是墨墨背单词软件，没有用任何纸质的书，以每天300个单词左右的量过完了第一遍大纲，5月份到8月份，快速过完了两本考研英语单词书，分别是红宝书和恋恋有词。在8月份到考前，就是尽可能多次的过核心词，到这个时候基本上大纲的词都可以很深刻的记住了。其次是语法，我当时看的是刘晓燕的语法课并结合她的长难句书做练习。阅读方面，我看了马天艺的阅读技巧课还有一些唐迟的课，我个人认为没有什么太大的帮助，有用的技巧很少，这个就见仁见智了。翻译上我是使用的田静的翻译真题书，把英一和二的历年真题都做了一遍并对照的答案。真题是9月份开始的，用的是张剑的黄皮书。我只做了英语二的历年真题，第一遍只做阅读，把不会的生词做好记录，第二遍刷真题就是掐时间做所有题。作文是背王江涛老师的作文范文，把书中给的十大范文背熟，最后结合背的范文写出自己的框架。

数学：
数学我跟的是张宇的高数全家桶，一些难点看了武老师的强化课单独对应的章节，跟了武老师的每日一题，李永乐大帝的线代。基础阶段是3月份开始，看张宇基础30讲的同时做300题，暑假之前结束基础阶段，强化阶段做张宇的1000、武的660和880，高数第一遍强化结束后，一边做高数的题集一边跟李永乐大帝学线代基础和强化，线代只做了660。9月中旬结束强化，开始二刷18讲，把所有例题搞懂（多刷教材很重要！）。11月开始做真题，从08年的开始，只做数二，两天一张。最后做的是李林的6+4预测卷和3张张宇老师的4套卷，但是可惜今年没压中题。

有一点我想说一下，根据今年的数学选择题来看，可能以后选择题一部分会倾向于考定义，定理，所以在学习数学的过程中一定要把定义定理扎扎实实的理解并记忆。

专业课：
877科目较少，内容简单，C语言如果有基础的话刷刷题即可，没基础的同学就去B站搜教程，一搜一大把，看播放最多的就行。数据库的话，除了sql和关系图以外都是要背的知识点，找学长画一下重难点或者买笔记就可以。历年录取的专业课分数大家都不低，可见真的是不压分。

一些坑

  • 最好不要报班，花钱找不自在，我周围报班的同学没有一个上岸的，其中不乏培训费大辄几万的机构，我个人认为考研最重要的就是自制力，自知力不够的话找个研友或者能够看得住你的人。资源网上都有，就算花钱买一些名师的课也比培训班强得多。
  • 所有科目的真题都不要着急做，做的早了很可能会导致你后面不想做了，这就很大程度的浪费了真题，最好在10月份到11月份的时间，自己的基础扎实了以后再着手上真题。

复试

复试没什么好说的，很公平，线上复试成绩占比最大的就是综合面试，这个是根据每个人的个人情况来定考试的内容的，只要好好准备没什么问题。

就说这么多，望诸君共勉。
